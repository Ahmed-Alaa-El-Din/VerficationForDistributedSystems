1. Install quicklisp : https://www.quicklisp.org/beta/
	 1. wget https://beta.quicklisp.org/quicklisp.lisp
	 2. sbcl --load quicklisp.lisp
	 3. (quicklisp-quickstart:install)
	 4. (ql:add-to-init-file)
	 5. (ql:quickload "quicklisp-slime-helper")
	 6. add to ~/.emacs.d/init.el those two lines:
					(load (expand-file-name "~/quicklisp/slime-helper.el"))
					(setq inferior-lisp-program "sbcl")
	 7. Evalute them : select them them M-x eval-region
	 8. M-x slime-restart-inferior-lisp
2. To run the application: ./start.lisp
3. To run from repl:
	 1. M-x slime-compile-and-load-file while in verifier.asd
	 2. (asdf:oos 'asdf:load-op 'verifier) in REPL
4. Coverage testing: compile and load coverage.lisp
			 sbcl --load coverage.lisp
5. Unit testing
			 sbcl --load test.lisp
